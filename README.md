### 사용한 Python 문법
### List 문법
* list(string): 한 문자씩 나누기([]배열)
* map(func,x): 리스트 x의 각 요소에 func을 적용
* set(list): list에서 중복된 값을 제거(set: 집합 자료형)
* len(list): list 요소 개수
